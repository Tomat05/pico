# pico

## Thonny

### Windows

- Go to the [website](https://thonny.org/) and download the Windows installer for the latest supported Python version 

### Mac

- Open the terminal and

```sh
brew install thonny
```

### Linux

- Arch

```sh
yay -S thonny
```

- Debian/Ubuntu

```sh
sudo apt install thonny
```

- Fedora

```sh
sudo dnf install thonny
```
